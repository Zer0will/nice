FRAMEWORKS STATEMENT

This project utilized the three main frameworks of HTML, CSS, and Javascript in the creation of this website. In adherence to the National TSA Guidelines, no AI generated code nor templates were used in the creation of this website.
The Bootstrap framework was used in the creation of this website, however we can confirm that under no circumstances did we use any templates, themes, or code that was not created by the team.
